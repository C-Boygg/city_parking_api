# City Parking API

RESTful API для сервиса управления городским парковочным пространством.

Проект разработан в рамках ознакомительной практики. Приложение позволяет хранить данные о парковочных местах, получать список мест, изменять статус занятости и обрабатывать ошибочные запросы.

## Стек технологий

- Python
- Django
- Django REST Framework
- SQLite
- JSON

## Возможности

- создание парковочного места
- получение списка парковочных мест
- получение свободных парковочных мест
- обновление статуса парковочного места
- получение статистики
- обработка ошибок в JSON-формате
